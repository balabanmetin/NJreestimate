------------------------------------
NJ reestimate
------------------------------------
NJ reestimation of tree branch lengths when data contains duplicates


------------------------------------
Requirements
------------------------------------
1. Python: Version >= 3.0
2. `treeswift`


------------------------------------
How to run
------------------------------------

`python run_NJreestimate.py -s data/aln_trimmed.fasta -t data/astral.lpp.nwk -o data/out.nwk`
